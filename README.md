## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd my-nodejs-webpage
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   node src/app.js
   ```

4. **Open your browser and navigate to:**
   ```
   http://localhost:3000
   ```
